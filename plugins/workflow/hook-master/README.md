# hook-master

Claude Code hooks toolkit — create, configure, and manage hooks across user, project, and local scopes.

## Installation

```
/plugin marketplace add eskygroup/esky-ai-knowledge-base
/plugin install hook-master@esky-flightcontent-marketplace
```

## Skills

| Skill | Description |
|-------|-------------|
| hook-master | Create, configure, list, remove, and explain Claude Code hooks |

## Usage

```
/hook-master:hook-master create PreToolUse
/hook-master:hook-master list
/hook-master:hook-master explain SessionStart
/hook-master:hook-master remove
```
